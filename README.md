## Why?

This project demonstrates the minimal setup required to call Rust code from Kotlin/Native, producing a native binary. It serves as a reference for integrating the two ecosystems without unnecessary boilerplate.

## Supported Platforms

- macOS (x64, arm64)
- Linux (x64, arm64)
- Windows (x64)
